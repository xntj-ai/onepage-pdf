HTML → PDF · 一张连续长页

把一个 HTML 页面,
变成一张连续长 PDF

onepage-pdf 是一个 Claude Code / Agent 技能。给它一个长滚动式 HTML 页面,它就产出一张超长的单页 PDF —— 不分页、不断裂,桌面版排版和背景原样保留,文字仍可选中。还能可选地脱敏字符串,并自带泄漏校验。

普通的"打印成 PDF"会把长页面切成一片片 A4,内容卡在分页线上断开。onepage-pdf 反其道而行:它从不预测打印高度。它先渲染到一张超大的页面上,从渲染好的 PDF 里量出内容真实底边,再裁切到刚好贴合 —— 所以高度天然精确。只要 Python 加一个本地 Chrome;不需要 Playwright,也没有要常驻的服务。

分页 vs 一张连续长页

同一个长页面,导出成两种样子 —— 这就是这个技能的全部意义。

普通打印成 PDF

被切成一摞 A4

  • 每个 A4 分页处内容被拦腰截断
  • 断点在约 741px 处触发,把网格压塌
  • px→pt 取整误差凭空多出一页
onepage-pdf

一张连续的长纸

  • 没有分页 —— 从头到尾一张纸
  • 桌面版排版与背景原样保留
  • 裁切到内容的精确高度

超大尺寸打印

无 Playwright 零服务 文字可选中

可选脱敏:--replace 在渲染前替换字符串,--forbid 在任一禁用词残留于 HTML 或最终 PDF 文字时中止。

核心特性

一张连续长页 · 永不分页

先渲染到一张超大的基岩页上,再把 MediaBox + CropBox 裁到内容真实高度。没有 A4 切口,不会从行中间截断。

高度是裁出来的 · 不是算出来的

打印高度从渲染好的 PDF 里量出,绝不靠屏幕 scrollHeight 去猜 —— 天然精确。

保住排版与背景

print-color-adjust: exact 留住渐变;固定背景被归一化;滚动渐显动画被强制显形。

脱敏 + 泄漏校验

渲染前替换字符串,并断言一份禁用词清单在 HTML 与最终 PDF 文字里都不残留。

适用场景

适合

要发一张干净的 PDF

  • 必须以一张 PDF 离开浏览器的 HTML 提案、报告、落地页
  • 用聊天软件或邮件发出去、要在手机上干净打开的交付件
  • 桌面上看着对的 AI 生成 PPVI 风格页面
不适合

真打印 / 像素级还原

  • 真的需要分页 A4 的打印任务 —— 正常打印即可
  • 会被打印路径栅格化掉的 WebGL / canvas / 透视场景
  • 两种裁切启发式都搞不定的页面(改走 CDP 路线)

快速开始

装进 Claude Code 技能目录

git clone https://github.com/xntj-ai/onepage-pdf.git ~/.claude/skills/onepage-pdf
pip install pymupdf

对 Claude 说,或直接跑脚本

说"用 onepage-pdf 把这份提案转成不分页的连续单页 PDF" —— Claude 会检查 HTML、写好断点锁定再转换。或直接调用:python scripts/onepage_pdf.py input.html -o output.pdf --width 1280。

肉眼核对结果

脚本打印 OK 1 page, WxHpt。渲染一张缩略图,确认排版完好、背景都在、底部没被截断。
